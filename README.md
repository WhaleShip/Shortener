# Shortener Webapp
This FastAPI application is designed to shorten and manage URLs. It also includes support for user authentication and authorization.

## Stack
- Python
- FastAPI
- SQLAlchemy
- Pydantic
- Alembic
- JWT
- bcrypt
- PostgreSQL
- PgBouncer


## Up instruction

### 1. Clone project
```commandline
git clone https://github.com/WhaleShip/BucketBot.git
```

### 2. Create .env file
```commandline
make env
```

### 3. Run service
```commandline
make autorun
```

<Br>

### To turn off the service
```commandline
make off
```
